• 🌟 Artificial Intelligence (AI)
    Goal: Make machines act smart like humans
    1950s → Present | Cycle 0 (Origin)
    • 🤔 Non-Learning AI
      Hardcoded rules, no learning
      1950s–1980s | Cycle 1
      • Rule-Based Systems
        IF this → DO that
      • Logic Engines
        Formal reasoning
      • Automation Scripts
        Repetitive tasks
      • ❌ Limitation
        No learning or adaptation
      • ✅ Leads to
        Machine Learning
    • 📈 Machine Learning (ML)
      Learns patterns from data
      1980s–2010s | Cycle 2
      • 🧩 Classical ML
        Humans design features
        • Feature Engineering
          Manual feature design
        • Tasks
          Regression, Classification
        • Models
          Decision Trees, SVM, k-NN
        • ❌ Limitation
          Fails on complex data
        • ✅ Leads to
          Deep Learning
      • 🧠 Deep Learning (DL)
        Learns features automatically
        2010s → Present | Cycle 3
        • 🔗 Neural Networks (Foundation)
          Core model behind deep learning
          • Input Layer
            Receives data
          • Hidden Layers
            Learn patterns
          • Output Layer
            Prediction
          • Weights & Biases
            Control learning
          • Activation Functions
            Non-linearity (ReLU, Sigmoid)
          • Loss Function
            Measures error
          • Forward Pass
            Data flows through network
          • Backpropagation
            Adjusts errors
          • Optimization
            Improves learning
          • 👁️ CNN
            Image understanding (edges → objects)
          • 🔁 RNN / LSTM
            Sequence data (time, text)
          • ⚡ Transformers
            Attention-based architecture
            Introduced ~2017 | Cycle 4
            • Attention
              Focus on important info
            • Self-Attention
              Relates inputs
            • Multi-Head Attention
              Multiple perspectives
            • 🧠 LLMs
              Built using Transformers
              2020s → Present | Cycle 5
              • Prompt
                Instruction + context
              • Tokenization
                Text → tokens
              • Embeddings
                Tokens → vectors
              • Transformer Layers
                Process relationships
              • Next Token Prediction
                Generate output
              • Training
                Pretraining, Fine-tuning, RLHF
          • 🚀 Applications
            Real-world usage of models
            • 🗣️ NLP
              Language understanding
            • 👀 Computer Vision
              Image & video analysis
            • 🧩 RAG
              LLM + external knowledge
