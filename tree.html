• 🌟 Artificial Intelligence (AI) [1950s → Now | Cycle 0]
    Goal: Make machines act smart like humans
    • 🤔 Non-Learning AI [Cycle 1]
      Hardcoded rules, no learning
      • Rule-Based Systems
        IF this → DO that
      • Logic Engines
        Formal reasoning
      • Automation Scripts
        Repetitive tasks
      • ❌ Problem
        No learning, no adaptation
      • ✅ Leads to ML
    • 📈 Machine Learning [1980s–2010s | Cycle 2]
      Learns patterns from data
      • 🧩 Classical ML
        Human-designed features
        • Feature Engineering
        • Regression / Classification
        • Decision Trees, SVM, k-NN
        • ❌ Limitation
          Fails on complex data
        • ✅ Leads to DL
      • 🧠 Deep Learning [2010s → Now | Cycle 3]
        • 🔗 Neural Networks (Foundation)
          • Input Layer
            Receives data
          • Hidden Layers
            Learn patterns
          • Output Layer
            Prediction
          • Weights & Biases
            Control learning
          • Activation Functions
            ReLU, Sigmoid
          • Loss Function
            Measures error
          • Forward Pass
            Data flows
          • Backpropagation
            Fix errors
          • Optimization
            Improve weights
          • 👁️ CNN
            Images → edges → objects
          • 🔁 RNN / LSTM
            Sequences (time, text)
          • ⚡ Transformers [2017 | Cycle 4]
            • Attention
            • Self-Attention
            • Multi-Head Attention
            • 🧠 Large Language Models [2020s | Cycle 5]
              • Prompt
                Instruction + context
              • Tokenization
              • Embeddings
              • Transformer Layers
              • Next Token Prediction
              • Training
                Pretraining, Fine-tuning, RLHF
              • 🛠️ Tools (Examples)
                ChatGPT, Claude, Gemini
        • 🚀 Applications
          • 🗣️ NLP
            Language understanding
          • 👀 Computer Vision
            Image/video AI
          • 🧩 RAG (Pipeline)
            LLM + external knowledge
            • User Query
            • Search (DB / Vector Store)
            • Retrieve Documents
            • Augment Prompt
            • LLM Generates Answer
