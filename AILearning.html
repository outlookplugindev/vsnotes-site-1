Artificial Intelligence – Complete Hierarchy

Author: Ashish Sinha · AI Developer 
◆ Artificial Intelligence (AI) [1956]
│
│  AI is the grand goal of making machines behave intelligently like humans.
│
│  Major AI Domains (Beginner Friendly):
│   ◇ NLP  – Natural Language Processing (text, chat, search)
│   ◇ Vision – Image & video understanding
│   ◇ Speech – Voice recognition, TTS
│   ◇ Robotics – Physical intelligent systems
│   ◇ Recommender Systems – Netflix, YouTube
│
│  Examples:
│   ▸ Chatbots
│   ▸ Google Maps
│   ▸ Face recognition
│
├── ◆ Rule‑Based / Symbolic AI
│   │
│   │  Humans explicitly write IF–ELSE rules.
│   │  No learning, no improvement.
│   │
│   │  Failed because:
│   │   • Rules explode in real life
│   │   • Cannot handle uncertainty
│
└── ◆ Machine Learning (ML) [1990s]
    │
    │  ML lets machines learn patterns from data.
    │
    │  Learning loop:
    │   Data → Prediction → Loss → Backpropagation → Better Model
    │
    ├── ◇ Learning Types
    │   ├─ Supervised Learning
    │   │   • Labeled data
    │   │   • Example: Spam vs Not‑Spam
    │   │
    │   └─ Unsupervised Learning
    │       • No labels
    │       • Example: Customer clustering
    │
    └── ◆ Deep Learning (DL) [2012]
        │
        │  ML failed on raw images, speech and text → DL started.
        │
        │  Uses Neural Networks inspired by human brain.
        │
        ├── ◇ Neural Network Layers
        │   ├─ Input Layer   → receives raw data
        │   ├─ Hidden Layers → learn patterns
        │   └─ Output Layer  → prediction
        │
        ├── ◆ CNN (Vision Models)
        │   │
        │   │  Example: Cat vs Dog
        │   │   Layer‑1 → edges
        │   │   Layer‑2 → textures
        │   │   Layer‑3 → parts (ears, eyes)
        │   │   Output  → Cat / Dog
        │
        └── ◆ Transformer [2017]
            │
            │  Uses Attention instead of memory loops.
            │  Handles long context efficiently.
            │
            └── ◆ Large Language Models (LLMs) [2018+]
                │
                │  Examples:
                │   ▸ ChatGPT
                │   ▸ Microsoft Copilot
                │   ▸ Google Gemini
                │
                │  Example Input: "I love you"
                │
                ├── ◇ Tokenization
                │
                │   Word Tokens:
                │    ["I", "love", "you"]
                │
                │   Sub‑word (BPE) Tokens:
                │    ["I", "lo", "ve", "you"]
                │
                │   Why BPE?
                │    • Handles unknown words
                │    • Reduces vocabulary size
                │
                ├── ◇ Vocabulary
                │   Fixed dictionary mapping token → ID
                │   Example:
                │     "I"    → 101
                │     "love" → 458
                │     "you"  → 932
                │
                ├── ◇ Embedding Layer (Vector Creation)
                │
                │   Token IDs → dense vectors
                │
                │   Example Embedding Matrix:
                │
                │    Token     d1    d2    d3    d4
                │    --------------------------------
                │    "I"      0.12  0.88 -0.31  0.05
                │    "love"   0.45 -0.72  0.61 -0.18
                │    "you"    0.09  0.33  0.77  0.22
                │
                │   Vectors capture meaning & similarity.
                │
                ├── ◇ Transformer Layers
                │
                │   Self‑Attention:
                │    "love" focuses on "I" and "you"
                │
                │   Feed‑Forward:
                │    Refines contextual meaning
                │
                ├── ◇ Vocabulary Projection
                │   Converts vectors → word probabilities
                │
                └── ◇ Generated Text
                    Highest probability token selected
                │
                └── ◆ RAG (Retrieval‑Augmented Generation)
                    │
                    │  Adds external knowledge to LLM.
                    │
                    ├─ Data Collection → Docs / PDFs
                    ├─ Data Cleaning
                    ├─ Chunking
                    ├─ Vector Embedding
                    ├─ Vector Database (Azure AI Search / FAISS)
                    ├─ User Prompt → Vector
                    ├─ Similarity Search
                    ├─ Prompt + Context
                    └─ Grounded LLM Answer
